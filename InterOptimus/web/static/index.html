InterOptimus · MatRIS MLIP（本地）

上传薄膜 / 基底 CIF，配置点阵匹配与结构参数；本机运行 run_simple_iomaker（calc = matris）。 完成后可查看报告、交互式极图，并下载各 pair 最优界面的 POSCAR 压缩包。

使用命令行启动时，首次运行会自动在 ~/.interoptimus/web_venv 创建独立 Python 环境并安装依赖，无需手动 conda activate。 仅打开浏览器、由服务器托管时，请使用 Docker（见仓库 Dockerfile）。
