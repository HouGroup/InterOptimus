InterOptimus GUI

请选择使用场景。三个界面分开运行：本地交互计算、新版任务管理（推荐）、以及旧版工作流 Hub。

Recommended

任务管理（Manage）

新版界面计算数据管理中心。提交 MLIP / MLIP+VASP 任务、查看任务列表与状态，点入任意任务可看到结构 3D 可视化、CIF 下载和分阶段进度。

  • 提交新任务（MLIP only / MLIP + VASP）
  • 统一任务库（MongoDB / 本地 JSON 自动回退）
  • 结构 3D 可视化 + CIF 下载
  • 分阶段进度：MLIP → 每个 match 筛选出 term → VASP
 Local

IOMaker Local

原来的本地浏览器图形界面。适合在打开浏览器的节点上直接跑 MLIP 任务、看实时日志和可视化。

  • 只支持本地 MLIP 计算
  • 不负责分配集群任务
  • 不作为长期数据库管理入口
 Legacy

IOFlow Hub（旧版）

初版界面计算工作流 / 任务库浏览器，仅有简洁列表 + 旁边详情面板。已被「任务管理（Manage）」取代，仍保留以便回看。

  • 面向 jobflow-remote / 多节点计算
  • 无提交表单 / 无结构可视化
  • 仅用作迁移参考
