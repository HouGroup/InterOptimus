Recommended

任务管理（Manage）

界面计算数据管理中心。提交 MLIP / MLIP+VASP 任务、查看任务列表与状态，点入任意任务可看到结构 3D 可视化、CIF 下载和分阶段进度。

  • 提交新任务（MLIP only / MLIP + VASP）
  • 统一任务库（MongoDB / 本地 JSON 自动回退）
  • 界面结构 3D 可视化 + CIF / 结果文件下载
  • 分阶段进度：MLIP → 每个 match 筛选出 term → VASP
 Local

IOMaker Local

本机交互式 GUI。适合在打开浏览器的节点上直接运行 MLIP 任务、查看实时日志与中间结果可视化。

  • 只支持本地 MLIP 计算
  • 不分配集群任务，不作为长期数据库
  • 实时日志、relax / final 可视化
